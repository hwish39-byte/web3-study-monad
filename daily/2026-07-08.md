# 2026-07-08 学习日志

## 今日学习内容

### 1. Web3 实习手册 — 安全与合规

阅读 [安全指南](https://web3intern.xyz/zh/security/)：

- **私钥 = 所有权**：谁掌握私钥谁就控制资产，绝不分享、截图或上传
- **助记词备份**：物理离线保存（纸笔），不要存在云盘/截图/聊天记录
- **合约风险**：approve 授权上限、钓鱼合约、蜜罐代币
- **最小权限原则**：合约交互时只授权所需额度，用完 revoke（使用 revoke.cash 等工具）
- **课程专用钱包**：用独立钱包做课程实验，主力资产放在冷钱包或单独地址

### 2. BuildAnything Freshman — 钱包 & 区块链概念

- **加密钱包**：管理私钥/地址的工具，EOA（MetaMask）vs 智能合约钱包（Safe, Argent）
- 钱包的核心功能：生成密钥对、签名交易、与 dApp 交互（通过 WalletConnect / 浏览器扩展）
- **交易**：from → to → value → data → nonce → signature
- **Gas**：Gas Limit × Gas Price = 交易费。Gas Limit 不够 = 交易失败（revert），但仍扣费

### 3. Monad Docs 阅读

- [Monad Docs](https://docs.monad.xyz/)：Monad 是高性能 EVM 兼容 L1，主打并行执行（Parallel Execution）
- [Testnets](https://docs.monad.xyz/developer-essentials/testnets)：Monad Testnet 信息，RPC 端点、Chain ID、浏览器
- [Block Explorers](https://docs.monad.xyz/tooling-and-infra/block-explorers)：Monadscan 用于查询交易、地址、合约状态

### 4. 以太坊官方文档

- [Wallets](https://ethereum.org/en/wallets/)：EOA 钱包类型、安全性分级
- [Transactions](https://ethereum.org/en/developers/docs/transactions/)：交易字段详解（nonce, gasPrice, gasLimit, to, value, data, v, r, s）
- [Gas](https://ethereum.org/en/developers/docs/gas/)：Gas 是 EVM 执行的计价单位，复杂操作（存储/合约创建）消耗更多

### Explorer 解读

一笔交易的典型字段：

```
Transaction Hash:  0x...         ← 唯一标识
Status:            Success        ← 成功 / Failed
Block:             1234567        ← 所在区块高度
Timestamp:         14 secs ago    ← 确认时间
From:              0x...          ← 发起方
To:                0x...          ← 接收方 / 合约地址
Value:             0.01 MON       ← 转账金额
Gas Used:          21000          ← 简单转账固定 21000
Gas Price:         0.1 Gwei       ← 单位 Gas 价格
Transaction Fee:   0.0000021 MON  ← Gas Used × Gas Price
```
