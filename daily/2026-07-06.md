# 2026-07-06 学习日志

## 今日学习内容

### 1. Web3 实习手册 — 入门导读 & 区块链基础

阅读 [Web3 实习手册](https://web3intern.xyz/zh/) 以下章节：

- **前言 / 入门导读**（https://web3intern.xyz/zh/preface/）
  - Web3 ≠ 炒币，核心是"所有权互联网"——用户拥有自己的数据、身份和资产
  - 从 Read-Only Web1 → Read-Write Web2 → Read-Write-Own Web3
  - Proof of Work 的两层含义：挖矿的 PoW，以及求职时的作品集 PoW（本 repo 就是）

- **区块链基础概念**（https://web3intern.xyz/zh/blockchain-basic/）
  - 区块链是分布式账本，每个区块包含交易记录 + 前一个区块的哈希 → 链式结构不可篡改
  - 去中心化：没有单一实体控制，节点共同维护
  - 共识机制：PoW（工作量证明）、PoS（权益证明）—— 以太坊已转 PoS
  - 区块浏览器：用来查交易、地址、合约状态（Etherscan / Monadscan）

- **以太坊概览**（https://web3intern.xyz/zh/overview-of-ethereum/）
  - 以太坊 = 区块链 + 智能合约平台（图灵完备）
  - 账户模型：EOA（外部账户，私钥控制）vs Contract Account（合约账户，代码控制）
  - Gas：执行操作的计费单位，Gas Price × Gas Used = 交易手续费
  - 交易结构：from, to, value, data, nonce, gasLimit, maxFeePerGas, etc.

## 摘录 / 笔记

> "Web3 is the internet owned by the builders and users, orchestrated with tokens." — Packy McCormick

关键术语表：

| 术语 | 含义 |
|------|------|
| 区块链 | 分布式、不可篡改的账本 |
| 区块 | 存储一组交易的结构，通过哈希链接 |
| EOA | 外部账户，由私钥控制，发起交易 |
| 智能合约 | 链上运行的代码，不可篡改 |
| Gas | 计算资源的计价单位 |
| 钱包 | 管理私钥、签名交易的工具（MetaMask） |
