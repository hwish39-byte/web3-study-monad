## 4. 用户参与活动后的下一步行动

### 主行动

活动结束后，用户需要在 7 天内完成一个 Builder 小任务，并将成果发布到社区或个人社交平台。

### 行动路径

1. 选择一个任务方向。
2. 用一句话写清楚任务目标。
3. 用 AI Agent 拆成 3-5 个执行步骤。
4. 在 7 天内完成一个可展示版本。
5. 发布成果，并写一句复盘：AI Agent 帮我完成了哪一步？

### 推荐任务方向

- 项目研究卡片：选择一个 Web3 / AI 项目，整理背景、问题、产品、用户和个人判断。

- 内容复盘：写一篇“我如何用 AI Agent 完成一次任务拆解”的文章。
