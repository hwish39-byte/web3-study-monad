# Web3 Study — Monad Buidler Camp

> 一个结构化的 Web3 学习旅程，聚焦 Monad 生态。
> 本仓库作为 **proof of work** —— 记录每天的学习日志、动手实验和任务产出，
> 涵盖 Monad Buidler Camp 及课外的 Web3 学习。

## 目录结构

```
web3-study-monad/
├── daily/           学习日志（每天一篇，记录学到的东西、踩的坑、总结）
├── tasks/           任务卡片（Monad Camp 的每个里程碑/任务拆成独立卡片）
├── experiments/     代码实验（Solidity 合约、Hardhat/Foundry 项目、前端 demo）
├── templates/       模板文件（daily-note.md / task-note.md，建新文件时直接复制）
└── README.md        本文件
```

## 目录说明

**daily/** —— 每天一篇 markdown，包含：今天学了什么、遇到的问题 & 解决思路、明日计划。

**tasks/** —— 每个任务独立成卡，含目标描述、技术方案、执行步骤、关键代码片段。

**experiments/** —— 可运行的代码文件，可以直接 clone 下来跑。包含合约源码、测试脚本、前端原型等。

**templates/** —— 模板文件。`daily-note.md` 供写每日日志时复制，`task-note.md` 供新建任务卡片时复制。
